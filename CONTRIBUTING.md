# Contributing

欢迎贡献具身智能相关资源。

## 可以贡献什么

- 论文
- 开源项目
- 数据集
- 课程
- 仿真环境
- 机器人平台
- 中文导读
- 复现笔记

## 推荐格式

```md
### Resource Name

- Link:
- Type:
- Topic:
- Why it matters:
```

## 收录标准

- 链接可访问
- 信息尽量来自官方页面、论文或代码仓库
- 有一句话说明价值
- 标注适合初学、进阶还是研究

## 不建议提交

- 无法访问的链接
- 只有标题没有说明
- 纯广告
- 与具身智能关系不大的泛 AI 内容
